### New/Changed/Fixed in "2017.04.12.16.00"
### Neu/Geändert/Behoben in "2016-10-09 20:00"
.) Changed: The default value for ClearEPG is now set to "true" to manage the new EPG behavior of Digital Media Server 2.0.0.0.
   Geändert: Der Standardwert für ClearEPG ist jetzt auf "true/wahr" gesetzt, um dem dem geänderten EPG-Verhalten des Digital Media Server 2.0.0.0 zurechtzukommen.
.) Changed: The option CreateEPGEntry no longer fills all "holes" in a channel's EPG. If there is no EPG at all, an entry for the next 24 hours is created. This is to manage the new EPG behavior of Digital Media Server 2.0.0.0.
   Geändert: Die Option CreateEPGEntry füllt nicht mehr alle "Löcher" im EPG eines Senders auf. Wenn überhaupt keine EPG-Einträge vorhanden sind, wird einer für die nächsten 24 Stunden erstellt. Dies passiert, um mit dem geänderten EPG-Verhalten des Digital Media Server 2.0.0.0 zurechtzukommen.

### Neu/Geändert/Behoben in "2016-10-01 14:00"
.) New: The option ClearEPG clears all existing EPG data before the EPG update is started.
   Neu: Die Option ClearEPG löscht alle bestehenden EPG-Daten bevor das EPG-Update gestartet wird.
.) Changed: CreateEPGEntry now creates one EPG entry for today and for each of the next seven days, each one lasting 24 hours.
   Geändert: CreateEPGEntry erstellt jetzt einen EPG-Eintrag für heute und für jeden der nächsten sieben Tage, jewils mit 24 Stunden Laufzeit.
